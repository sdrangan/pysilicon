# Installing PySilicon

PySilicon requires **Python 3.10 or newer**. Pick the path that fits you:

- **[User Setup](./users.md)** — *use* PySilicon as a library in your own project.
  `pip install` from GitHub into a virtual environment; no clone needed. Start here if
  you just want to build designs with PySilicon.
- **[Developer Setup](./developers.md)** — *modify* PySilicon, contribute changes, or
  run the bundled examples and tests. Clone the repository and install in editable mode
  with the development tools.

The optional AI-assistant integrations — MCP server, OpenAI-backed semantic example
search, and the VS Code extension — are covered separately under
[AI Tooling](../ai_tooling/). None of them are required to use or develop PySilicon.
