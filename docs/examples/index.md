# Examples

PySilicon includes example projects that demonstrate how the framework’s architectural model, DataSchema system, and simulation environment come together in practice. These examples are designed to be small, readable, and directly connected to the concepts introduced in the overview documentation.

The examples in this section mirror the structure of the `examples/` directory in the repository. Each example includes both runnable Python code and narrative documentation that explains the design choices and how PySilicon components interact.

---

## Available Examples

- [WaveFlow](./waveflow):  A reconfigurable wireless processor with a tileable architecture
and dynamic flow creation.

Additional examples will be added as the framework evolves.
